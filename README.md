# SingleSpa 子应用

## 技术栈

- vue cli 5
- vite
- element-plus
- typescript
- sass
- tsx
- eslint 

## 运行

### Vite SingleSpa 启动

```shell
npm run vite
```

### Vite 单独启动

```shell
npm run vite:standalone
```

### Webpack SingleSpa 启动

```shell
npm run webpack
```

### Webpack 单独启动

```shell
npm run webpack:standalone
```

### 代码格式化

```shell
npm run lint
```
